# generate

Generates code from a service definition.

```
POST /generate
{
  "definition": { "name": ... },
  "generator": { "name": ... }
}
--- 200 OK
{

  "failure": { "line": ... }
}
```

| request | type | description |

| definition | [NamedText](NamedText.md) | The service definition. |

| response | type | description |
